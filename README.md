# 🧮 TkinterCalculator

A simple and user-friendly **GUI Calculator** built using **Python Tkinter**. This project performs basic arithmetic operations through an interactive graphical interface, making it a great beginner project for learning Python GUI development.

---

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🧹 Clear (C) button
- 🖥️ Simple and clean graphical user interface
- ⚡ Fast and responsive calculator
- 🚫 Handles division by zero by displaying an error

---

## 🛠️ Technologies Used

- Python 3
- Tkinter (Python GUI Library)
- Visual Studio Code
- Git & GitHub

